# Advanced IoT Weather Station 
This project is a comprehensive weather monitoring system designed to measure environmental parameters in real-time.

# Features
* **Multi-Sensor Integration:** Monitors Temperature, Humidity (DHT22), and Air Quality (MQ135).
* **Wind Speed Analysis:** Custom algorithm to calculate wind speed via Anemometer (0-30 m/s).
* **OLED Interface:** Data visualization using SSD1306 OLED display with a menu system.
# Tech Stack
* C++ / Arduino
* I2C Communication Protocol
* Analog Signal Processing
